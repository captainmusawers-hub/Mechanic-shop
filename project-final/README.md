# Surrey Centre Auto Repairs Inc. — Website

Professional auto repair website for **Surrey Centre Auto Repairs Inc.**, located at 13654 104 Ave, Surrey, BC.

## Features

- Full responsive single-page website (Vite + React + TypeScript + Tailwind CSS v4)
- Sticky animated header with mobile menu
- Hero section with background image
- Services catalog (Mechanical, Maintenance, Specialty Tech, Safety)
- Specialty service highlights (Key Programming, Luxury/German vehicles)
- About section
- **Tire Search Widget** — search by tire size or by vehicle (Year/Make/Model)
- Photo gallery with hover effects
- Customer reviews accordion
- Appointment booking form
- Floating live chat widget
- Fully accessible (ARIA labels, semantic HTML, keyboard navigation)
- SEO meta tags

## Getting Started

**Prerequisites:** Node.js 18+

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
# → http://localhost:3000

# Build for production
npm run build

# Preview production build
npm run preview
```
